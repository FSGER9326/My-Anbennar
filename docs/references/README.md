## How to use it

These references are for:

- syntax reminders
- trigger/effect lookup
- system-specific EU4 modding rules

They are not the first source of truth.

Use this trust order:

1. existing Anbennar repo pattern
2. local reference snapshots in this folder
3. live wiki or other primary source if the local snapshot is incomplete

## Maintenance notes

- Keep links relative so this folder works on all environments.
- If a crosswalk references a wiki topic that is not yet snapshotted locally, mark that gap in `reference-index.md`.
