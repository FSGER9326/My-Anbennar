# Verne ID Ledger

This ledger is the single place to track custom Verne IDs so future updates do not accidentally collide, drift, or silently break.

## Naming rules (required)

- **Required prefix:** all new Verne IDs must start with `verne_`.
- **Suffix conventions:**
  - `_mod` for modifiers
  - `_trg` for scripted triggers
  - `_eff` for scripted effects
  - `_idea` for idea groups/ideas
  - `_reform` for government reforms
  - `_evt` for event IDs/namespaces or event key stems
  - `_dec` for decisions
  - `_pol` for policies
  - `_flag` for country/province/global flags
  - `_var` for variables
  - `_loc` for localization key stems
- **Retired ID rule:** never reuse IDs marked `deprecated`; mint a fresh `verne_` ID instead.

## Status values

- `planned`: reserved or designed, not yet live in game scripts.
- `active`: implemented and expected to be used.
- `deprecated`: retired; keep for history only, never reuse.
