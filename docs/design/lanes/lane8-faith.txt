# Status: PARTIALLY IMPLEMENTED — 6 missions across 3 slots

# ===============================================
# Lane 8 - Faith & Apostolic Empire (positions across slots, early→late)
# ===============================================
# This lane forms Verne's religious backbone — from early Corinite
# consolidation through HRE faith dominance to the Holy Corinite Empire
# and world faith emperor capstone.
#
# The lane has three major arcs:
# 1. Early:   Corinite defence → religious expansion (slot 3, pos 6-8)
# 2. Mid:     Faith-war synthesis — Corinite military integration (slots 4-5)
# 3. Late:    Holy Corinite Empire → World Faith Emperor (slot 4, pos 12+)
#
# Missions that fit this lane:
# Slot 3: A33_corins_shield (pos 6), A33_spread_the_word (pos 7),
#         A33_in_the_name_corin (pos 8)
# Slot 4: A33_the_holy_corinite_empire (pos 12)
# Slot 5: A33_a_crimson_sea (pos 9), A33_religious_mercantilism (pos 10)

# --- Actual missions implemented ---

# A33_corins_shield (pos 6, slot 3)
# Requires: none (standalone)
# Trigger: Religion corinite, is defender of faith
# Effect: Converts vernman provinces to corinite (50% chance), stability, inquisitor advisor
# Commentary: The faith lane opener. Verne claims the mantle of Corin's shield —
# the defender of the faith. The 50% conversion chance on vernman provinces
# is a powerful early-game religious consolidation tool.

# A33_spread_the_word (pos 7, slot 3)
# Requires: A33_corins_shield
# Trigger: DIP 200, religious unity 0.90, stability 2, province 43 (corinite),
#          provinces 380/367 owned by corinite or trusted allies/subjects
# Effect: Fires verne.104 event on target nations, prestige
# Commentary: The missionary expansion mission. Requires both domestic religious
# unity and external Corinite allies, pushing Verne to actively spread the faith
# beyond its borders through diplomacy or conversion events.

# A33_in_the_name_corin (pos 8, slot 3)
# Requires: A33_the_lands_of_adventure (Lane 2), A33_spread_the_word
# Trigger: 3 colonial nations with 10 provinces each, theologian/inquisitor 4,
#          H67 opinion 100 (or not exist), province 6559 (corinite, dev 25, shipyard, manpower)
# Effect: Corinite syncretism in Taychend (oren_nayiru provinces get verne_corinite_syncretism),
#         colonial_lai_peninsula claims, creates Port of Adventure at 6559
# Commentary: The overseas faith mission — projecting Corinite religion into the
# New World through colonial nations. The syncretism mechanic for Oren Nayiru
# provinces is particularly interesting — Verne doesn't destroy local faith but
# weaves Corinite practice into it. The Port of Adventure at 6559 ties faith
# expansion to the adventure network.

# A33_the_holy_corinite_empire (pos 12, slot 4)
# Requires: A33_the_vernman_era (Lane 3 convergence)
# Trigger: Legitimacy 90, power projection 75, prestige 90, all HRE nations corinite
#          with reform passed, OR is Z01
# Effect: world_network +1, verne_corinite_halann (permanent), 30 trade power for
#         propagate religion, verne_refunding_holy_order_15 flag, changes country color
# Commentary: The HRE faith lock — every nation in the Empire must be Corinite.
# This is the hardest faith trigger in the tree. The permanent modifier and
# trade power bonus make Verne a religious-economic hegemon. The color change
# (to crimson) is a visual statement of faith dominance.

# A33_a_crimson_sea (pos 9, slot 5)
# Requires: A33_united_under_crimson_wings (Lane 4)
# Trigger: Akasik region all subject-held, corinite religion, accepted Akasi cultures,
#          treasury 500, CoT level 2 in 402/400/382
# Effect: overseas_projection +1, world_network +1, verne_imperial_maritime_court (10yr),
#         enables Aur-Kes-Akasik monument, converts subject religion, reduces liberty desire
# Commentary: The overseas faith-commerce mission. Converting the Akasik coast to
# Corinite trade networks. Shared with Lane 5 (Trade) — the CoT requirements
# ensure this is as much about commerce as faith.

# A33_religious_mercantilism (pos 10, slot 5)
# Requires: A33_a_crimson_sea
# Trigger: 16 merchants, 50 heavy ships, 150 light ships
# Effect: Enables propagate religion in trade companies, verne_trade_company_conversion,
#         inquisitor advisor, verne_grand_fleet_tier1 (permanent)
# Commentary: The religious-commerce fusion. Verne's trade fleet becomes a conversion
# engine — every trade company province slowly turns Corinite. Shared with Lane 5 (Trade).
