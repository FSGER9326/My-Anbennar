# ===============================================
# Lane 9 - Industrial Foundries (Slot 2/8, early→late)
# ===============================================
# This lane forms Verne's industrial backbone — from the Khenak resource
# extraction through Red Brass forge development to controlled devastation
# doctrine and industrial military production.
#
# The lane has two source streams:
# 1. Slot 2 (early): A33_the_riches_of_the_khenak (pos 2) — resource foundation
# 2. Slot 8 (mid-late): A33_red_brass_forge (pos 20), A33_controlled_devastation (pos 22)
#
# The Khenak foundry system uses a scripted path (verne_overhaul_seed_khenak_foundry_path)
# that is seeded by multiple missions and checked by later content.

# --- Actual missions implemented ---

# A33_the_riches_of_the_khenak (pos 2, slot 2) [IMPLEMENTED]
# Requires: A33_alvars_reforms
# Trigger: Province 838 (Khenak capital) dev 15, base production 5;
#          Province 292 (Heartspier) dev 15, base production 5;
#          Province 291 (Calasandur) dev 15, base production 5;
#          Treasury 200
# Effect: Seeds khenak_foundry_path, unlocks merc_khenak_talons (free), -200 treasury,
#         verne_redbrass_surplus (20yr), workshops in 838/292/291,
#         conditional: best dev province of 377/364/6513 gets verne_khenak_excavations
# Commentary: The industrial lane opener. Khenak is Verne's resource heartland —
# the red brass mines and metalwork that fuel the foundry economy. The conditional
# excavation bonus rewards developing the highest-potential Khenak province.
# The merc_khenak_talons company ties industrial output directly to military force.

# A33_red_brass_forge (pos 20, slot 8) [IMPLEMENTED]
# Requires: A33_the_riches_of_the_khenak
# Trigger: Khenak foundry path seeded, manufactories institution or any manufactory,
#          furnace in 838/292/291, treasury 500
# Effect: -500 treasury, verne_red_brass_forge (15yr), +0.02 army professionalism,
#         +25 MIL
# Commentary: The mid-lane industrial milestone. Building a furnace in the Khenak
# heartland converts raw resource extraction into advanced metallurgy. The Red Brass
# Forge modifier provides sustained military-economic bonuses — army professionalism
# and direct MIL power — rewarding the infrastructure investment.

# A33_controlled_devastation (pos 22, slot 8) [IMPLEMENTED]
# Requires: A33_red_brass_forge
# Trigger: Khenak foundry path seeded, battle_evocation or red_court doctrine OR
#          khenak_foundry flag + MIL 100, at war or won war against A30/A01 in 10yr
# Effect: -100 MIL, verne_controlled_devastation (15yr), -10% artillery cost,
#         +0.5 yearly manpower
# Commentary: The late industrial capstone. Controlled devastation doctrine uses
# industrial output (foundries, artillery) to overwhelm enemies — the artillery
# cost reduction and manpower bonus represent the foundry-to-army pipeline at
# full capacity. The war trigger ensures this can only fire during active conflict.

# --- Khenak foundry path system ---
#
# The khenak_foundry_path is a scripted state variable seeded by:
# - A33_the_riches_of_the_khenak (primary seeder)
# - A33_in_search_of_adventure (Lane 2, secondary seeder)
# - A33_the_rogue_duchy (Lane 3, checks if seeded for bonus)
#
# Checked by:
# - A33_red_brass_forge (this lane)
# - A33_controlled_devastation (this lane)
# - A33_the_rogue_duchy (Lane 3, grants integration bonus if seeded)
#
# Scripted trigger: verne_overhaul_has_khenak_foundry_seeded
# Scripted effect: verne_overhaul_seed_khenak_foundry_path

# --- Cross-lane dependencies ---
# - A33_the_riches_of_the_khenak gates A33_the_grand_port_of_heartspier (Lane 1)
# - A33_the_riches_of_the_khenak also gates A33_the_rogue_duchy (Lane 3) bonus path
# - A33_red_brass_forge requires Lane 2 secondary seed (adventure system)
# - A33_controlled_devastation requires either Red Court doctrine (Lane 6) or
#   the foundry flag — creating a choice between arcane and industrial paths
