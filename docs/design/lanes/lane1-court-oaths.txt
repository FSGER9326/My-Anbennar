# Verne Overhaul - Lane 1: Court & Oaths
# Slot 1 — Verne's cultural-diplomatic foundation, the Vernissage,
# the Halanni world network, and the Project Holohana capstone.
# Part of the 10-lane mission tree (10-column blueprint, 2026-04-01).
# Status: IMPLEMENTED — all 9 missions present in missions/Verne_Missions.txt

# ===============================================
# Lane 1 - Court & Oaths (Slot 1, early→late)
# ===============================================
# Missions in A33_first_slot forming the cultural-diplomatic backbone.
# The Vernissage system (Port of Adventure network) runs through this lane,
# establishing Verne as a network state projecting soft power worldwide.

# --- Actual missions implemented ---

# A33_the_vernman_renaissance (pos 2)
# Trigger: Corinite religion, renaissance embraced, 3 advisors or culturally diverse advisors
# Effect: Institution spread to Heartspier + Calasandur, splendor, potential general (Alvar IV)
# Modifier: escanni_center_of_learning (province, 25yr)
# Commentary: Opens the Verne playthrough by requiring cultural-religious modernization.
# This is the founding mission of the entire tree — no required_missions.

# A33_the_grand_port_of_heartspier (pos 3)
# Requires: A33_the_riches_of_the_khenak, A33_the_vernman_renaissance
# Trigger: Heartspier (288) dev 25, base production 7, trade building, CoT level 2, expanded infra
#           Moorhalls (376) CoT level 2
# Effect: Creates Ports of Adventure at 288 and 376, seeds estuary companies path,
#         advances overseas_projection, grants verne_new_armada (25yr) + wyvern gulf (20yr, Alvar IV)
# Commentary: The first real sea-state mission. Establishes the maritime infrastructure
# that feeds both the Vernissage system and the Adventure Network.

# A33_the_vernissage (pos 6)
# Requires: A33_the_grand_port_of_heartspier
# Trigger: Province 292 (Vernissage) expanded infra x2 or dev 30, marketplace, edict,
#          level 3 artist or level 4 ADM advisor, 1000 treasury, 200 DIP, 3 adventures completed
# Effect: Advances world_network, fires advisor event + verne.105 delayed event
# Commentary: The Vernissage is Verne's unique cultural institution — a university-museum-hotel
# that draws international visitors. This mission establishes it as a network node.

# A33_expand_the_Vernissage (pos 8)
# Requires: A33_the_lands_of_adventure
# Trigger: Province 292 dev 40, edict, level 3 advisors in all categories, 3000 treasury,
#          5 adventures completed
# Effect: Advances world_network, upgrades to verne_vernissage_tier2 (permanent),
#         conditional bonuses from infrastructure/economic/admin ideas
# Commentary: Deepens the Vernissage into a mature institution. The idea-group conditional
# bonuses reward players who invested in the right idea groups alongside the Vernissage.

# A33_zenith_of_the_eastern_princes (pos 9)
# Requires: A33_expand_the_Vernissage
# Trigger: Province 4963 (Taychend hub) fully built up (manpower, fort, forcelimit, shipyard,
#          infra x2 or dev 30, most trade power), 10 provinces in lupulan_rainforest_region
# Effect: Fires verne.110 event
# Commentary: Verne's eastern colonial capstone — consolidating the Lupulan Rainforest
# and establishing a dominant trade hub in Taychend.

# A33_the_halanni_exposition (pos 10)
# Requires: A33_zenith_of_the_eastern_princes
# Trigger: Province 292 dev 50, prosperous, edict, level 5 advisors all categories,
#          3000 treasury, 200 DIP, dip rep 6, 20 countries with 100+ opinion, 8 adventures
# Effect: Fires verne.111, grants verne_halanni_exposition_bureau (3yr), advisor event,
#         conditional bonuses from diplomatic/influence/court ideas
# Commentary: The Halanni Exposition is Verne's world's fair — a global cultural event
# that cements Verne as the world's premier network state.

# A33_the_heart_of_darkness (pos 11)
# Requires: A33_the_halanni_exposition
# Trigger: Province 2219 has Port of Adventure + 40 units present, 40 provinces in effelai_superregion
# Effect: Fires verne.1004
# Commentary: Verne's deepest overseas reach — the Effelai Heart expedition.
# A late-game colonial capstone requiring massive military commitment.

# A33_the_grand_vernissage (pos 12)
# Requires: A33_the_heart_of_darkness
# Trigger: Province 292 infra x4 or dev 60, 40 cavalry in province, level 5 advisors all,
#          10000 treasury, 10 adventures completed
# Effect: Fires verne.112, advisor event, scaled rewards based on Vernissage relic collection
#         (15 relics, each adding ADM/DIP/MIL power, prestige, splendor, innovativeness)
# Commentary: The Vernissage Grand Opening is the ultimate cultural achievement.
# The relic-collection scaling makes this one of the most rewarding missions in the tree.

# A33_project_holohana (pos 14)
# Requires: A33_the_grand_vernissage, A33_all_roads_lead_to_verne
# Trigger: 300 ADM/DIP/MIL, mage tower in 292, 30000 treasury, 13 adventures completed
# Effect: Fires verne.128
# Commentary: The final capstone of the Court & Oaths lane. Project Holohana is Verne's
# ultimate mega-project — a world-changing initiative that requires mastery of every system.

# --- New modifiers (referenced from common/event_modifiers/verne_overhaul_modifiers.txt) ---
#
# verne_new_armada              — +navy tradition, +naval morale (25yr, from Grand Port)
# verne_halanni_exposition_bureau — +1 dip rep, +1 merchant, +0.5 prestige (3yr)
# verne_vernissage_tier1/tier2  — province modifiers on 292 (permanent)

# --- Cross-lane dependencies ---
# - A33_the_grand_port_of_heartspier gates A33_across_the_pond (Lane 2)
# - A33_the_vernissage gates A33_the_lands_of_adventure (Lane 2)
# - A33_expand_the_Vernissage gates A33_vernissage_secretariat (Lane 5, new content)
