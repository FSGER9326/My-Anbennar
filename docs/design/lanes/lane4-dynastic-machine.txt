# ===============================================
# Lane 4 - Dynastic Machine (Slot 4, mid→late)
# ===============================================
# This lane forms Verne's imperial spine — the path from duchy to empire.
# It covers naval military buildup, Busilar conquest/alliance, HRE emperor
# mechanics, and culminates in the Holy Corinite Empire and artificery
# wyvern technology.
#
# The lane has two major arcs:
# 1. Mid:     Naval buildup → Busilar conquest → Union of Crowns → HRE emperor
# 2. Late:    Faith lock → Sword & Shield → Holy Empire → Artificery wyverns

# --- Actual missions implemented ---

# A33_taking_to_the_seas (pos 3) [IMPLEMENTED]
# Requires: A33_a_matter_of_pride (Lane 3)
# Trigger: Navy size >= A29, 10 heavy ships, 2 naval leaders, 7 port provinces
#          with shipyard/dock/high production
# Effect: Claims on Busilar (if not allied), verne_impressment_drive (25yr),
#         glorious_engagement naval doctrine (or navy tradition + sailors)
# Modifier: verne_impressment_drive (25yr)
# Commentary: The naval military opener for the dynastic lane. Compares
# directly against rival A29's navy size, creating a competitive dynamic.

# A33_taming_the_lion (pos 4) [IMPLEMENTED]
# Requires: A33_taking_to_the_seas, A33_the_wyvern_nest_initiative (Lane 5)
# Trigger: Busilar areas fully owned OR fully owned+allied with A47,
#          level 3 mil advisor or level 2 theologian/commandant
# Effect: Claims on businor_region (or split with A47), army tradition,
#         splendor, potential admiral, trust with A47
# Commentary: The Busilar conquest is one of Verne's defining mid-game
# moments. The alliance vs conquest choice creates a meaningful split.

# A33_a_union_of_crowns (pos 5) [IMPLEMENTED]
# Requires: A33_taming_the_lion, A33_across_the_pond (Lane 2), A33_the_kingdom_of_Verne (Lane 3)
# Trigger: 200 DIP, 50 power projection, Businor fully controlled (with or
#          without A47 alliance, trust 80+), no unrest
# Effect: +1 world_network, claims on isle_of_tef, verne_wingspan (25yr),
#         fires verne.106 (crowns event), potential A47 vassal bonuses
# Modifier: verne_wingspan (25yr)
# Commentary: The Union of Crowns is the dynastic culmination of the Busilar
# arc. The three-path trigger (conquest, alliance, vassal) creates branching.

# A33_united_under_crimson_wings (pos 8) [IMPLEMENTED]
# Requires: A33_spread_the_word (Lane 3), A33_the_might_of_the_wyvern (Lane 5),
#           A33_a_union_of_crowns
# Trigger: Emperor or Z01, 7 specific provinces with forcelimit buildings,
#          supply depots in 4 areas (if defavored adventurers) or 100 MIL
# Effect: 15 imperial influence, +10/+10 ADM/DIP for 20 corinite HRE members,
#         upgrades wyvern_nest_expanded → wyvern_nest_grand (permanent)
# Commentary: The HRE consolidation mission. Requires emperor status and
# military infrastructure across the heartland. The wyvern nest upgrade
# transforms temporary military bonuses into permanent infrastructure.

# A33_corins_devout_protectors (pos 9) [IMPLEMENTED]
# Requires: A33_united_under_crimson_wings, A33_in_the_name_corin (Lane 3)
# Trigger: HRE corinite locked or (emperor + aristo_revoke/imperial_dynasty),
#          DoF tier 3, 400 corinite provinces
# Effect: verne_dynasty_protected_court (2yr), dynasty_protected flag,
#         merc_crimson_wake_lances, verne_only_dotf global flag,
#         verne_special_cb_enabled flag, removes ravelian societies (50%),
#         verne_no_imperial_rectorate flag, fires verne.116
# Commentary: Locks the HRE to Corinite faith and removes Ravelian influence.
# The global flag verne_only_dotf prevents other nations from becoming DoF.
# This is an irreversible mechanical commitment.

# A33_with_sword_and_shield (pos 10) [IMPLEMENTED]
# Requires: A33_corins_devout_protectors, A33_a_crimson_sea (Lane 5)
# Trigger: DoF tier 5, 300 army size, highest dev in western Cannor,
#          14 corinite african countries OR 500 corinite african provinces
# Effect: +1 world_network, verne_apostles_of_corin (permanent),
#         verne_apostolic_oceanic_command (3yr)
# Modifier: verne_apostles_of_corin (permanent), verne_apostolic_oceanic_command (3yr)
# Commentary: The faith-war capstone. Requires total military and religious
# dominance in both Cannor and Africa.

# A33_the_holy_corinite_empire (pos 12) [IMPLEMENTED]
# Requires: A33_the_vernman_era (Lane 3)
# Trigger: 90 legitimacy, 75 power projection, 90 prestige, all HRE members
#          corinite + aristo_revoke/imperial_dynasty (or Z01)
# Effect: +1 world_network, prestige if maritime court active,
#         30 trade power for propagate religion flag, verne_refunding_holy_order_15,
#         verne_corinite_halann (permanent), color change (if Z01)
# Modifier: verne_corinite_halann (permanent)
# Commentary: THE holy empire capstone. Every HRE member must be Corinite.
# The color change to crimson (if Z01) is a visual representation of the
# Corinite world order.

# A33_type_2_wyverns (pos 13) [IMPLEMENTED]
# Requires: A33_on_wings_of_artificery (Lane 5), A33_the_holy_corinite_empire
# Trigger: 25000 treasury, damestear or precursor relic trading bonus,
#          artificers+mages loyalty 60+, mithril province (10 dev, production,
#          manufactory) owned/subject/ally
# Effect: +2 dynastic_magic_machine, ends verne_red_court_scandal disaster
#         if active, fires verne.120
# Commentary: The artificery-magic fusion capstone. Type 2 Wyverns represent
# the pinnacle of Verne's arcane-military technology — bio-engineered war
# mounts powered by damestear and mithril.
