# ===============================================
# Lane 7 - Military Orders (positions across slots, mid→late)
# ===============================================
# This lane forms Verne's holy-military backbone — the path from early military
# pride through imperial unification to holy order dominance and martial supremacy.
#
# The lane currently has no dedicated slot block. Missions that fit this theme
# are spread across slots 3, 4, and 5:
#
# Slot 3: A33_a_matter_of_pride (pos 2) — military pride / territorial consolidation
# Slot 4: A33_united_under_crimson_wings (pos 8) — military unification under faith
#         A33_corins_devout_protectors (pos 9) — holy order / defender of faith
#         A33_with_sword_and_shield (pos 10) — faith warfare capstone
# Slot 5: A33_the_might_of_the_wyvern (pos 7) — military power projection
#         A33_break_the_queen_of_the_hill (pos 1) — military conquest opener
#
# The verne_knights_of_the_crimson_scale_reform (in verne_overhaul_reforms.txt)
# exists in the codebase but is not yet gated by any mission. This lane is the
# natural home for it.

# --- Actual missions that fit this lane ---

# A33_a_matter_of_pride (pos 2, slot 3) [IMPLEMENTED]
# Requires: A33_alvars_reforms, A33_old_friends_old_rivals, A33_break_the_queen_of_the_hill
# Trigger: damespearl_area all owned, prestige 40, power projection 25, DIP 50
# Effect: Fires verne.101
# Commentary: The territorial consolidation mission that establishes Verne's military
# confidence. Securing the Damespearl is a statement of martial intent — the first
# step toward military order status. Shared with Lane 3 (Maritime Empire).

# A33_break_the_queen_of_the_hill (pos 1, slot 5) [IMPLEMENTED]
# Requires: none
# Trigger: Treasury 100, MIL 50, 364 owned+cored, khenak_area owned+cored, 377 has 10 units
# Effect: Fires verne.102
# Commentary: The military conquest of the Khenak gnolls — establishing Verne's
# martial credibility in the homeland. Shared with Lane 5 (Trade & Colonisation)
# as the gateway to the wyvern economy.

# A33_united_under_crimson_wings (pos 8, slot 4) [IMPLEMENTED]
# Requires: A33_spread_the_word, A33_the_might_of_the_wyvern, A33_a_union_of_crowns
# Trigger: Is emperor or Z01, forcelimit buildings in 7 key provinces, supply depots
#          in wyvernmark/khenak/hapiande/busilari_straits (if defavored adventurers)
# Effect: +15 imperial influence, upgrades wyvern_nest_expanded → wyvern_nest_grand,
#         +10 MIL/DIP for 20 Corinite HRE nations
# Commentary: The imperial unification mission — Verne consolidates its military
# orders under the crimson banner of the emperor. The supply depot requirement
# (for defavored adventurer path) rewards military infrastructure investment.

# A33_corins_devout_protectors (pos 9, slot 4) [IMPLEMENTED]
# Requires: A33_united_under_crimson_wings, A33_in_the_name_corin
# Trigger: HRE religion locked corinite OR is emperor with reform, defender of faith tier 3,
#          400 corinite provinces
# Effect: verne_dynasty_protected_court (20yr), sets verne_dynasty_protected flag,
#         unlocks merc_crimson_wake_lances, sets verne_only_dotf (global),
#         sets verne_special_cb_enabled, removes ravelian societies, fires verne.116
# Commentary: The holy order capstone — Verne becomes the sole defender of the
# Corinite faith and claims special CBs against heathens. The Crimson Wake Lances
# mercenary company is the military order made manifest.

# A33_with_sword_and_shield (pos 10, slot 4) [IMPLEMENTED]
# Requires: A33_corins_devout_protectors, A33_a_crimson_sea
# Trigger: Defender of faith tier 5, army size 300, highest dev in western Cannor,
#          14 corinite African countries OR 500 corinite African provinces
# Effect: world_network +1, verne_apostles_of_corin (permanent),
#         verne_apostolic_oceanic_command (10yr)
# Commentary: The faith warfare capstone. Verne's military orders now project
# power across continents — the Apostles of Corin modifier represents a permanent
# holy military presence worldwide.

# A33_the_might_of_the_wyvern (pos 7, slot 5) [IMPLEMENTED]
# Requires: A33_the_sea_nest
# Trigger: Land forcelimit 150, naval forcelimit 100, 20 hussars/40 cavalry,
#          2 generals, 2 conquistadors, 2 admirals, 2 explorers, power projection 50+
# Effect: Fires verne.109, verne_might_of_the_wyvern (permanent), unlocks merc_heartspier_skyguard
# Commentary: Military power projection through the wyvern cavalry corps and the
# Heartspier Skyguard mercenary company. The ultimate expression of Verne's
# military orders — airborne knights riding wyverns into battle.
