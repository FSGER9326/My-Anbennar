This folder documents how existing Anbennar systems are actually implemented in the repository.

Use these files when you want:

- real repo anchors before editing
- examples of how a system is coded in practice
- a quick map of which files belong to one gameplay feature
- a reminder of where Anbennar differs from a simpler vanilla-style implementation
