## Delta from vanilla-style baseline

- large custom rebel catalog
- many system-specific rebel identities
- stronger content coupling between rebel types and custom mechanics/events

## Verne adaptation note

When designing pressure/disaster outcomes, validate whether an existing rebel type already represents the intended failure mode.
